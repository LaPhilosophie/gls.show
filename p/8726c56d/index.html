使用zotero高效管理文献

本文最后更新于：2024年1月13日 凌晨

简介

zotero是一个优秀的科研文献管理工具。它开源、免费、跨平台，适配了Windows、macos、Linux。在可以对文献进行高效操作的同时，支持通过使用浏览器插件对网页进行收录操作

简单总结一下zotero：

  • 提供文献的高效化分类整理
  • 支持翻译功能
  • 支持markdown语法的笔记功能
  • 支持

在zotero的官网上，有这样一篇文章：Why Zotero?，讲述了为什么使用zotero

下载并安装zotero：https://www.zotero.org/

基本使用

导入文献

条码导入

支持输入ISBN、DOI、PMID、arXiv ID 或 ADS 条码来添加条目到您的文库，并且支持多行导入

直接拖动文件导入

把本地pdf文件拖进去即可

使用浏览器插件导入

安装浏览器插件zotero connector，可以将网页内容保存到zotero中

复制文献信息

在zotero中选中条目，c+shift+c即可

复制效果：

同步功能

在zotero官网上注册账号，在软件菜单栏-》首选项-》同步处登录账号，即可实现多设备同步

不同账号可以使用Zotero群组功能，开一个共享文献库，这样彼此读过的文献、作出的批注，也都可以共享

Zotero的群组功能的使用教程：https://zhuanlan.zhihu.com/p/348626383

在大目录显示所有文章（包括小目录的文章）

菜单-》查看-》勾选“显示子分类中的条目”

文章重要性标记

有一个很简单的方法对文章重要性进行排序，就是在论文标题列旁边显示版权信息即可，然后手动给论文的版权信息那一栏加上标记，比如###为重要，##为次要，#为一般，不加标记为不重要

插件

安装

打开工具-》附加组件，安装插件

  • Zotero Connector，打通和浏览器的屏障
    • https://www.zotero.org/download/connectors
  • 对参考文献进行高效管理
    • https://github.com/MuiseDestiny/zotero-reference/
  • 翻译功能
    • https://github.com/windingwind/zotero-pdf-translate/releases

翻译功能

GitHub - windingwind/zotero-pdf-translate: Translate PDF, EPub, webpage, metadata, annotations, notes to the target language. Support 20+ translate services.

支持多种翻译引擎，比如cnki、google、deep L、彩云小译、GPT等（可导入api）

以彩云小译翻译api为例：

  • 在彩云开放平台注册并登录
  • 打开应用管理-》创建应用-》翻译

记录笔记

zotero自带笔记功能的加强版

GitHub - windingwind/zotero-better-notes: Everything about note management. All in Zotero.

超链接快速跳转到zotero

zutilo支持通过超链接从别的软件，比如typora，导航到zotero文献的具体位置

Zutilo的高级扩展使用 Quick Copy alt1 alt2-CSDN博客

牛叉！Zutilo插件的这个功能，究竟有多好用！

论文重命名

很多时候下载下来的论文名字是混乱的，可以使用GitHub - jlegewie/zotfile: Zotero plugin to manage your attachments: automatically rename, move, and attach PDFs (or other files) to Zotero items, sync PDFs from your Zotero library to your (mobile) PDF reader (e.g. an iPad, Android tablet, etc.), and extract PDF annotations.自动的对文献进行重命名

中文文献支持

GitHub - l0o0/jasminum: A Zotero add-on to retrive CNKI meta data. 一个简单的Zotero 插件，用于识别中文元数据

中文期刊附件（PDF/CAJ）的元数据抓取，支持添加PDF/CAJ时自动抓取元数据功能

使用坚果云对zotero进行同步

zotero自带的云存储只有300MB，坚果云则是上传1GB，下载3GB，且支持webdav

同步方法：

  • zotero中，编辑-》首选项-》同步-》登录zotero的账号
  • 之后下面会出现文件同步方式，选择webdav
  • 注意下方的密码不是坚果云的密码。需要注册坚果云账号， 在坚果云网页端依次选择：账号名称——账户信息——安全选项——第三方应用管理——添加应用密码，给zotero生成一个密码
